# Sample Project

## Usage

### Change your payment pointer.

`public/index.html`

```html
<meta name="monetization" content="YOUR_PAYMENT_POINTER">
```

### Check Vue file.

`src/components/VueWebMonetizationTest.vue`

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```
